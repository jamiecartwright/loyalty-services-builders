# Setup

Configuration files and documentation for setting up this workspace.

## Contents

| File | Description |
|------|-------------|
| [MCP_SETUP.md](./MCP_SETUP.md) | MCP server configuration guide |
| [mcp.json.example](./mcp.json.example) | Template for `~/.cursor/mcp.json` |

## Quick Start

```bash
# Copy MCP config template to user settings
cp setup/mcp.json.example ~/.cursor/mcp.json

# Edit with your credentials
code ~/.cursor/mcp.json

# Restart Cursor
```
